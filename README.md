# Charles AI Trading Bot

Thank you for subscribing to Charles.

## Setup

1. Create a DigitalOcean droplet (Ubuntu 22.04, 2GB RAM minimum)
2. SSH into your droplet
3. Upload this folder via scp
4. Copy .env.example to .env and fill in your keys
5. Run: bash install-gate.sh https://charles-bot.xyz
6. Run: charles-activate
7. Run: charles-start

## Dashboard
http://YOUR_DROPLET_IP:8501

## Commands
charles-start    — start the bot
charles-stop     — stop the bot
charles-restart  — restart the bot
charles-logs     — view live logs
charles-status   — check status
charles-activate — register your server IP
